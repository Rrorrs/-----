# Anime Parser
### Функции 
Есть поиск по жанрам в который можно добавлять новые жанры в `template.html` в таблице с жанрами.

### Источники
Anime go, YummiAnime - Аниме, iconer - иконки, Warren davies - фон, Tailwind - кнопки, google fonts - шрифт `Monsterrat`.
